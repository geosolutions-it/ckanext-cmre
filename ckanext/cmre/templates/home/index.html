How to register

The CMRE Geo Data Fusion Centre Geo-Spatial Web Portal is a platform for scientific data hosting. It also supports collaboration and data exchange between CMRE and research partners. It hosts more than 10 years of CMRE sea trials, conference pages, and Scientific Web Applications. Accces to this Web Portal is password protected (except for a few public sections) and registration is required.

How it works

Once logged in, the user can browse applications, sea trials and conferences from the table of content on the left. The user will only see accredited sections (see paragraph above for new section request). FTP access at the URL ftp://geos.nurc.nato.int is available for a few sections and for a restricted period.
