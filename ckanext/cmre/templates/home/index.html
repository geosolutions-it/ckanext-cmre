How to register

Click on the link on the top-right of the page to request an user account. A Point of Contact at NURC must be specified in the request form. For new credential requests (e.g. access to new sections) please send an email to geos-webmaster@nurc.nato.int , specifying the username. Password recovery functionality is available from the login form. Please note that users of the previous edition of the NURC Geo Data Fusion Centre Web Portal (geos2.nurc.nato.int) must re-register.

How it works

Once logged in, the user can browse applications, sea trials and conferences from the table of content on the left. The user will only see accredited sections (see paragraph above for new section request). FTP access at the URL ftp://geos.nurc.nato.int is available for a few sections and for a restricted period.
